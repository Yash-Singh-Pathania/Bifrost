How it works

1

Import a video

Drop any video file into Bifrost, or click Import to browse. MP4, MKV, WebM, MOV, and more.

2

Automatic indexing

Bifrost extracts audio, transcribes it with Whisper, samples frames, and builds a searchable index with CLIP and Ollama.

3

Search by meaning

Type a natural language query. Bifrost searches both the transcript and visual content, ranked by semantic similarity.

4

Jump to the moment

Click any result to seek directly to that timestamp. Done.

Quick start

# Clone and install
git clone https://github.com/YashSingPathania/Bifrost.git
cd Bifrost && npm install

# Set up ML environment
npm run setup:python
ollama pull nomic-embed-text

# Run
npm run dev
